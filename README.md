# Angel Number Calculator

生年月日からあなたのエンジェルナンバー（ライフパスナンバー）を計算するツール。

## Live

https://mochibullet.github.io/angel-number/

## 計算方法

生年月日の全桁を足し合わせ、1桁（1-9）またはマスターナンバー（11, 22, 33）になるまで繰り返す。

**例: 1990年8月15日**
```
1 + 9 + 9 + 0 + 0 + 8 + 1 + 5 = 33 (Master Number)
```
